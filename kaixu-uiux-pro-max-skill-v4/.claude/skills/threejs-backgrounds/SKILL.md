# ThreeJS Backgrounds

Keep background layers modular and swappable. Never couple background animation to core UI logic. Provide reduced-motion fallback. Preserve mobile performance. Keep controls readable above the background. Do not block clicks or focus.

Prefer `src/components/backgrounds/`, `src/components/visuals/`, or `src/styles/backgrounds.css`. For static HTML, use CSS-only unless explicitly asked for JS/ThreeJS.
