# Smoke Checklist

✅ Build command exists or absence is reported.
✅ Frontend renders.
✅ Navigation links resolve.
✅ Buttons trigger real actions.
✅ Forms submit/export/store data as designed.
✅ Panels scroll and collapse/expand if implemented.
✅ Mobile viewport does not hide critical controls.
✅ No placeholder text remains in client-facing UI.
✅ Console has no obvious fatal runtime errors.
✅ No route was removed accidentally.
