# No-Theater Implementation

Search for TODO, FIXME, placeholder, demo, mock, fake, stub, sample, coming soon, not implemented, hardcoded success, hash-only links, and buttons without handlers.

Fix each issue by implementing real behavior, removing the visible claim/control, loud-failing with clear configuration error, or truthfully labeling local/sample mode.

Do not call pending manual payment a full checkout. Do not call header presence full webhook verification. Do not call file-existence smoke end-to-end proof.
