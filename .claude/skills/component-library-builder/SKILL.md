# Component Library Builder

Use existing component conventions. Preserve props/state/handlers. Keep components accessible. Make overflow regions scrollable. Make panel controls functional. Keep CSS tokens aligned with existing brand.

Do not introduce a second component system if one already exists. Do not add shadcn/Tailwind to static HTML without explicit conversion request.
